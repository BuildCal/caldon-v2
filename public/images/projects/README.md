# Project Images

Upload your project images here.

## Recommended Naming

- `project-slug.jpg` — Thumbnail for grid (800x600px)
- `project-slug-hero.jpg` — Hero image (1920x1080px)
- `project-slug-1.jpg`, `project-slug-2.jpg`, etc. — Gallery images

## Recommended Sizes

- **Thumbnails:** 800 x 600px (3:2 ratio)
- **Hero Images:** 1920 x 1080px (16:9 ratio)
- **Gallery Images:** 1200 x 900px (4:3 ratio)

## Format

- Use JPEG for photos
- Optimize images before upload (compress to ~80% quality)
